1、获取管理员帐户：向贵州茅台酒销售有限公司专管科咨询，获取专卖店管理员帐户。联系电话： 0852 2385049

2、登录网址： 用管理员帐户登录 http://www.emaotai.cn:90/zmd ， 注意：用管理员帐户登录，只能进行配置操作，不能办理业务 。

3、建立用户帐户： 使用管理员帐户登录后，必须为营业员、会计、保管员等人员建立帐户，并授予相应人员“营业员、保管员、会计、经理”等角色(可以一人担当多个角色)。

4、设置店面品种： 在商品设置功能中，加入本店销售的商品品种，期初数可以设置为０，以后可采用入库单、销售单调平帐目。

5、输入商品价格： 系统中预定义了８类商品价格，专卖店可以根据自己的需要选择价格类型，输入品种价格。

6、业务办理： 退出系统，重新进入，用营业员帐户登录，开始办理日常业务，如：商品入库、销售开票、库存管理等。
